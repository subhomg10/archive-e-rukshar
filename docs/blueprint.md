# **App Name**: Rukshar's Archive

## Core Features:

- Cinematic Entry: Minimalist landing page with a hero 'Enter' sequence featuring a subtle background glow and refined typography.
- Themed Poem Gallery: An 'All Poems' archive with theme filters and a premium search bar using fluid animations for grid transitions.
- Literary Reading Mode: A dedicated, distraction-free reader interface with generous line-height and calm fade-in content blocks.
- Supabase Integration: Connects to a Supabase backend to dynamically fetch and filter poem content, including titles, verses, and themes.
- AI Insight Tool: A feature that uses a generative AI tool to provide thematic analysis or emotional 'color' summaries for each poem.
- Micro-interaction Engine: Premium Framer Motion animations for button scaling, hover lift effects on cards, and animated nav underlines.

## Style Guidelines:

- Dark Theme: Primary color is a muted Champagne Gold (#C9B28F). Background is a Deep Onyx (#121113). Accent is a Soft Lavender (#948BB0).
- Headline and Body pairing: 'Alegreya' (Serif) for poetic text and titles to provide a literary feel; 'Inter' (Sans-serif) for navigational and functional UI elements.
- Minimalist grid with large white space and single-column center-aligned readers for high focus and zero distraction.
- 300ms smooth transitions with premium easing (e.g., [0.22, 1, 0.36, 1]) for all interactions, plus page-slide fade effects.
- Thin-weight stroke icons (e.g., Lucide or Feather) to maintain a lightweight, sophisticated archive aesthetic.
